Hello;World

Hello World ，中文意思：你好，世界。因为《The C Programme Language》中使用它做为第一个演示程序，非常著名，所以后来的程序员在学习编程或进行设备调试时延续了这一习惯。


  • 五种语言的Hello World;


  • JavaScript:

    console.log("Hello;World");

  • Php:

    echo "Hello;World";

  • C++

    cout<<"Hello, World!"

  • C

    printf("Hello, World!");

  • Python

    print("Hello, World!")
